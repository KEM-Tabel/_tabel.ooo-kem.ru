Авторизация в системе

1
Откройте письмо от отдела кадров с паролем
В письме вы найдете ваш пароль
2
Откройте страницу авторизации
Для начала работы с системой перейдите на страницу входа.
3
Введите учетные данные
Заполните поле "Пароль" вашим паролем.
4
Нажмите кнопку "Войти"
После ввода данных нажмите кнопку для завершения авторизации.
Подсказка

Если вы забыли пароль, обратитесь в отдел кадров для его восстановления.

Работа с табелем

1
Выбор месяца
В верхней части страницы используйте календарь для выбора нужного месяца.
2
Поиск сотрудников
Введите фамилию или имя сотрудника в строку поиска для быстрого нахождения.
3
Фильтрация по организациям
Нажмите кнопку фильтрации в правом верхнем углу, чтобы отобразить только сотрудников определенной организации.
4
Добавление комментариев
Нажмите правой кнопкой мыши на ячейку и выберите "Добавить комментарий" для внесения пояснений.
5
Документ 1с для заблокированных ячеек
Для просмотра документа 1с, необходимо навести курсор на заблокированную ячейку
6
Просмотр статистики
Для просмотра статистики, необходимо навестись курсором на строку пустых в правом углу Начальника, Мастера, или Объекта
7
Общий просмотр статистики
Для просмотра общей статистики по вашим сотрудникам, необходимо навестись курсором на Ваше Фамилия Имя Отчество в верхней части страницы

Ввод данных

1
Выберите сотрудника
Найдите нужного сотрудника в табеле или через поиск.
2
Откройте форму ввода
Дважды кликните по ячейке дня, в который нужно внести данные.
3
Выберите тип отметки
В появившемся меню выберите подходящий тип отметки (явка, отпуск, командировка и т.д.).
4
Ввод часов
Выделите нужную ячейку и введите количество часов с помощью клавиатуры.
5
Добавьте комментарий (при необходимости)
Используйте контекстное меню (правой кнопкой мыши) для добавления пояснений к отметке.
6
Редактирование заблокированной ячейки
Для редактирования заблокированной ячейки, необходимо нажать на неё левой кнопкой мыши, высветится окно редактором блокированной ячейки.
7
Система автоматически сохраняет изменения при выделении другой ячейки
Система автоматически сохраняет изменения при выделении другой ячейки, за исключением случаев, когда вы редактируете заблокированную ячейку.

Информация о сотруднике

1
Найдите сотрудника
Используйте поиск или найдите сотрудника в общем табеле.
2
Откройте карточку
Используйте соответствующую кнопку для просмотра детальной информации.
3
Просмотр данных
В карточке сотрудника вы увидите основную информацию
Подсказка

Некоторые данные могут быть доступны только для просмотра без возможности редактирования.

Другие функции

1
Доступ к справке
Нажмите кнопку "Помощь" в правом верхнем углу для получения дополнительной информации.
2
Выход из системы
Для завершения работы нажмите кнопку "Выход" в меню пользователя.
3
Защита от несвоеременного ответа от сервера при вводе данных
Если при вводе последующих данных, ответ от сервер не пришел, то система не даст ввести данные, пока не будет получен ответ от сервера, при этом появится уведомление в правом верхнем углу
Важно

Система оптимизирована для работы на компьютерах. На маленьких экранах смартфонов некоторые функции могут быть недоступны или работать некорректно.

Выделение ячеек имеет несколько вариантов

1
Выделение ячеек через мышь
1. Есть возможность выделить ячейку простым кликом левой кнопкой мыши
2. Есть возможность выделить группу ячеек, для этого необходимо нажать на первую ячейку и протянуть мышь до последней ячейки
2
Выделение ячеек через клавиатуру
1. Есть возможность перемещения выделения по ячейкам, для этого необходимо нажать нажимать на стрелки на клавиатуре
2. Есть возможность выделить группу ячеек, для этого необходимо на клавиатуре нажать shift выделить первую ячейку и с зажатой клавишей shift выделить последнюю ячейку
3. Есть возможность выделить ячейки в разном месте, для этого необходимо нажать нажать ctr и нажать на ячейку, с зажатой клавишей ctr выделить последующие ячейки
